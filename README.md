# Desafio 1 - Otimizador de Rotas de Entrega

Projeto desenvolvido para solucionar a rota de um caminhão de entregas utilizando a estratégia do Vizinho Mais Próximo.

## Arquitetura
O código foi construído utilizando Orientação a Objetos (POO) e refatorado seguindo as diretrizes de Clean Code e PEP 8
